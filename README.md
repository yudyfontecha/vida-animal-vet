### 🚀 Cómo abrir en VS Code

1. Abre VS Code
2. `File > Open Folder` → selecciona la carpeta `vida-animal`
3. Instala la extensión **Live Server** (si no la tienes)
4. Click derecho en `index.html` → **Open with Live Server**
5. El sitio se abrirá en tu navegador

- **Colores:** Edita las variables CSS en `css/styles.css` (líneas 1-30)
- **Textos:** Edita directamente en `index.html`
- **Fuentes:** Cambia los Google Fonts en el `<head>` del HTML
- **Imágenes:** Reemplaza los archivos en `images/`
